## Run Locally

1) Clone the project

```bash
  git clone https://github.com/MoniqueHeusinger/Movie-Database-PCC.git
```

2) Go to the project directory

```bash
  cd movie-database-pcc/frontend
```

3) Install dependencies

```bash
  npm install
```

4) Start the server

```bash
  npm run dev
```
